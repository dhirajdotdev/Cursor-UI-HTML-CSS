# Cursor UI Clone (HTML/CSS)

A pixel-perfect clone of the [Cursor.com](https://cursor.com) landing page, recreated using pure HTML and CSS.

## Project Links
- **Live Website:** [https://cursor-ui.netlify.app/](https://cursor-ui.netlify.app/)
- **Repository:** [https://github.com/dhirajdotdev/Cursor-UI-HTML-CSS](https://github.com/dhirajdotdev/Cursor-UI-HTML-CSS)

## Recreated Sections

The following sections from the original Cursor website have been faithfully recreated:

1.  **Navigation Bar**: Full responsive header with logo, navigation links (Features, Enterprise, Pricing, Resources), and CTA buttons.
2.  **Hero Section**: Main headline, download buttons, and the complex Editor screenshot mockup.
3.  **Editor Interface**: A detailed CSS-only recreation of the VS Code-like interface, including:
    *   Sidebar with "In Progress" and "Ready for Review" tabs.
    *   Tabbed file view (`index.html`, `style.css`).
    *   Syntax-highlighted code editor view using HTML/CSS.
4.  **Trusted Companies**: Grid of logos for companies using Cursor (Stripe, OpenAI, Midjourney, etc.).
5.  **Agent IDE Section**: "Agent turns ideas into code" feature showcase with deep UI mockups.
6.  **Feature Highlights**:
    *   **Tab**: "Magically accurate autocomplete".
    *   **Pull Requests**: "In every tool, at every step".
7.  **Testimonials**: Grid layout of reviews from industry leaders (Diana Hu, Jensen Huang, Andrej Karpathy, etc.).
8.  **Frontier Section**: "Stay on the frontier" cards showcasing models, codebase understanding, and security.
9.  **Changelog**: A list of recent updates and versions (e.g., 2.4, CLI updates).
10. **Join Us**: Recruitment section calling for engineers.
11. **Recent Highlights**: Blog post cards.
12. **Footer**: Comprehensive site map, social links, and legal information.

## Fonts & Colors

### Fonts
- **Primary Font**: `CursorGothic` (Custom font via `@font-face`)
- **Fallbacks**: `system-ui`, `-apple-system`, `BlinkMacSystemFont`, `"Segoe UI"`, `Roboto`, `sans-serif`

### Color Palette
The design uses a Dark Theme with distinct card elevations and subtle text contrasts.

**Core Colors:**
- **Background**: `#14120b`
- **Foreground (Text)**: `#edecec`
- **Secondary Text**: `#d7d6d5`, `#edecec99`

**UI Elements:**
- **Card Backgrounds**: `#1b1913`, `#1d1b15`, `#201e18`, `#26241e`, `#2b2923`
- **Card Hover**: `#201e18`
- **Borders**: `#edecec1a`

**Accents:**
- **Additions (Green)**: `#1f8a65`
- **Deletions (Red)**: `#cf2d56`
